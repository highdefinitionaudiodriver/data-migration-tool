# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.x   | ✅ |
| < 0.1   | ❌ |

## 脆弱性の報告

セキュリティ上の問題を発見した場合は、**Issue や Pull Request での公開報告は避け**、以下のメールアドレスへ直接ご連絡ください。

- **連絡先**：highdefinitionaudiodriver@gmail.com
- **対応 SLA**：ベストエフォート（72時間以内に初動連絡を目指します）

報告に含めていただけると助かる情報：

- 影響を受けるバージョン
- 再現手順
- 想定される影響範囲（PII 漏洩・データ破損・権限昇格・誤変換等）
- 可能であれば修正案
